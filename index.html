🔗 Apps Script URL
Googleスプレッドシートに紐付けたApps ScriptをデプロイしてできるURLを貼り付けてください。
📋 設定手順
  1. Googleスプレッドシートを新規作成する
  2. メニュー「拡張機能」→「Apps Script」を開く
  3. このページに添付された apps-script.js の内容を貼り付けて保存
  4. 「デプロイ」→「新しいデプロイ」→ 種類「ウェブアプリ」
  5. アクセスできるユーザー：「全員」に設定してデプロイ
  6. 表示されたURLをこの画面の入力欄に貼り付けて保存
📤 メンバーへの共有方法
このHTMLファイルを最初の1回だけLINEやメールで送ります。
以降はスプレッドシートにデータが保存されるため、
全員が↻ボタンを押すだけで最新データを取得できます。
アプリ自体を更新したときだけ再配布してください。
